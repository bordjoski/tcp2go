# tcp2go
tcp2go is port scanner. A fast one.


## Usage
    tcp2go scan <host>    



## Installation
tcp2go is not published module. To install it, run the following

    git clone https://github.com/bordjoski/tcp2go.git
    cd tcp2go  
    go install
